# Models Folder

This project uses the Gemini API for story generation when `GEMINI_API_KEY` is configured.

For college demos without an API key, `app.py` automatically falls back to a small rule-based NLP story generator so the workflow still works offline except for translation and gTTS network calls.

You can also use this folder to store notes, prompt experiments, or locally downloaded HuggingFace model files if you extend the project later.
